# Assets Guide

Place project images here.

## Gallery Photos

Add photos to:

```text
src/assets/gallery/
```

Supported formats:

```text
.jpg
.jpeg
.png
.webp
.avif
```

The React app automatically imports matching images from `src/assets/gallery` and displays them in the photo gallery. If the folder has no supported photos, the site shows romantic placeholder cards instead.

Suggested names:

```text
01-first-photo.jpg
02-favorite-smile.jpg
03-special-day.webp
```
